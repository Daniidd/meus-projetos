O que é HTML? ▼

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ele descreve a estrutura de uma página web usando marcações.

O que é CSS? ▼

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de documentos escritos em HTML ou XML. Ele controla o layout de múltiplas páginas web ao mesmo tempo.

O que é JavaScript? ▼

JavaScript é uma linguagem de programação que permite a implementação de funcionalidades complexas em páginas web, como conteúdos dinâmicos, interativos e controlados pelo usuário.

O que são funções de callback em JavaScript? ▼

Funções de callback são funções passadas como argumento para outra função, que são invocadas dentro da função externa para completar algum tipo de rotina ou ação.
